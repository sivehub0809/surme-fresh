Data principles

Built around trust, not data resale.

No data selling User data is not sold. Integrations receive only what is needed to execute the task.
Encrypted storage Sensitive chats, tokens, drafts, bookings, and pending actions use encrypted fields.
Row-level security Profiles, messages, memories, tasks, schedules, and tokens are scoped per user.
User control Users can disconnect integrations and delete their account data.
